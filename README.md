# ProyectoGrado

## Campamentos XR 
El analisis de los campamentos Xr se realizo en el archivo llamado Analisis_CampXR.ipynb. En este archivo se lee el archivo de participantes Camp Xr de excel el cual reune todos los participantes de los cursos y se homogenizo su formato. Es decir tienen las mismas categorias y en el mismo formato todos. Para correr entonces el .ipynb se tiene que descargar el excel y cambiar la ruta de este archivo en la segunda cacilla de codigo. Se debe correr el script completo para realizar la carga de datos y la generacion de las graficas. 


## Coursera y Laboratorios 

El analisis de estas dos secciones se hizo en el mismo archivo llamado Coursera_Y_Talleres.ipynb . En este archivo se leen 4 archivos de coursera con toda la informacion de los participantes: 
- users_creando_proy_xrYmv
- users_desarollo_mv_p_t
- users_desarollo_XR_unity
- users_RealidadExtendida_MV_paraTodos
Con esos archivos se corre toda la parte de coursera. Y para los talleres la infromacion esta ingresada manualmente por las fechas de los laboratorios. Para la parte del mosaico se tiene que dar el path de la imagen que se quiere recrear. 
Para correr el script se debe ejecutart el jupyter notebook completo. De esta manera se cargararan los datos,  se crearan las graficas y el mosaico.
